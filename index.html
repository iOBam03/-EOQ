สูตรที่ใช้คำนวณ (ปรับตามบริบทอสังหาฯ)
Economic Order Quantity
EOQ = √(2 × D × S / H)
D = อุปสงค์/ปี
S = Ordering Cost · ต้นทุนสั่งซื้อ (รวมค่าเอกสารจัดซื้อและค่าขนส่งต่อเที่ยวเข้าไซต์งาน)
H = Holding Cost · ต้นทุนถือครอง (สำหรับธุรกิจอสังหาฯ คิดจาก: ค่าเงินจมหน้างาน, ค่าความเสียหายพัสดุจากสภาพอากาศ และค่าสูญหายหน้าไซต์งาน)
Reorder Point
ROP = (D / 365) × Lead Time
จุดสั่งซื้อใหม่เพื่อไม่ให้ของขาด · หน่วยตรงกับสินค้า (ถุง / ชิ้น / เส้น ฯลฯ)
ต้นทุนบริหารคลัง (Inventory Mgmt Cost)
TC = (D/Q)S + (Q/2)H
เฉพาะต้นทุนสั่ง + ต้นทุนถือครอง (ไม่รวมมูลค่าสินค้า เพื่อให้เห็นส่วนต่างชัดเจน)
